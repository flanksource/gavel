React Grab → gavel todo

Capture any UI element into a gavel todo. Grab an element with React Grab, then run the Add to gavel todo action — a dialog opens with this gavel's /todos/new form prefilled with the component and its source.

Bookmarklet

Drag this to your bookmarks bar, then click it on any running dev page:

Add to gavel todo

Or paste in the console

var s=document.createElement('script');s.src='__GAVEL_ORIGIN__/react-grab-plugin.js';document.body.appendChild(s);
